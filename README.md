# ShiftMate (シフト管理ツール)

ShiftMateは、従業員のシフト提出と管理者のシフト調整をスムーズに行うためのWebアプリケーションです。ブラウザ上で動作し、データはローカルストレージに保存されるため、特別なサーバー構築なしですぐに利用を開始できます。

## 🌟 主な機能

### 管理者向け機能 (`index.html`)
- **ダッシュボード**: 従業員数、提出数、確定数、不足日などの状況を一目で確認可能。
- **従業員管理**: 従業員の追加および削除。
- **シフト管理**: 提出されたシフトの確認と「確定」処理。確定の取り消し（元に戻す）機能。
- **おすすめ機能**: 特定の日に出勤可能なスタッフの中から、勤務回数が少ないスタッフを優先して提案。
- **不足日アラート**: 平日・休日ごとの必要人数（昼2名・夜3名など）に対し、不足している日を自動でリストアップ。
- **給与計算（公平性レポート）**: 
  - スタッフごとの「平日」と「休日・祝日」の時給を個別に設定可能。
  - シフトの時間帯に基づいた労働時間を自動計算。
  - 給与締め日（毎月15日締め）に対応した月別給与の算出。
- **勤務回数グラフ**: 各スタッフの確定シフト数を視覚的に表示。

### 従業員向け機能 (`shift.html`)
- **シフト提出**: 自分の名前、日付、希望する時間帯、出勤/休みを選択してシフトを提出。
- **提出済みシフトの確認・削除**: 自分が提出したシフトの一覧確認や、誤って提出した際の削除機能。

## 🚀 使い方

1. 本プロジェクトのファイル（`index.html`, `shift.html`, `style.css`, `script.js`）を同じフォルダに配置します。
2. ブラウザで `index.html`（管理画面）を開きます。
3. **初期設定（管理者）**:
   - 「従業員登録」からスタッフの名前を入力し追加します。
   - 「時給設定」で各スタッフの平日・休日の時給を入力して保存します。
4. **シフト提出（従業員）**:
   - 上部のナビゲーションから「シフト提出 (`shift.html`)」に移動します。
   - 自分の名前を選び、日付・時間帯・出退勤ステータスを入力して提出します。
5. **シフト確定（管理者）**:
   - `index.html` に戻り、「提出状況一覧」からシフトを確定します。必要に応じて「おすすめ」機能を利用して勤務回数のバランスを取ることができます。

## 🛠 使用技術

- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- LocalStorage (データ保存用)

## 📝 注意事項
- データはご利用のブラウザのローカルストレージ（LocalStorage）に保存されます。ブラウザのキャッシュやデータをクリアすると、登録した従業員やシフトデータも消去されるためご注意ください。
- 祝日判定には固定のリスト (`holidays` 配列) が使用されています。年が変わる際などには適宜 `script.js` の日付を更新してください。
